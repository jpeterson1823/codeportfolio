Arch Installation notes:

Following install wiki:
	@pacstrap step:
		pacstrap /mnt base linux linux-firmware nano sudo dhcpcd networkmanager


Install drivers:
	Search available drivers:
		pacman -Ss xf86-video

	Install according driver:
		pacman -S package-name

	Install environment managers:
		pacman -S xorg gdm
		systemctl enable gdm
		pacman -S gnome [gnome-utils]

	Reboot and login to desktop environment.
